# 貸款精算小助手 (Loan Calculator)

這是一個基於 Web 技術開發的貸款試算工具，提供２種還款方式，旨在提供使用者直觀、快速的還款計畫分析。

## 🚀 專案特色

-   **雙重還款模式**：支援「本金平均攤還」與「本息平均攤還」。
-   **詳盡分析**：除了計算每月還款額，還提供總還款金額、總利息支出，以及完整的分期明細表。
-   **現代化 UI**：簡潔的介面設計與清楚的資料呈現。
-   **響應式設計 (RWD)**：支援手機與桌面裝置瀏覽。
-   **直覺化操作**：採用簡單的表單輸入 + 單選按鈕，符合使用者習慣。
-   **即時更新 (互動設計)**：輸入或切換還款方式後，會立即重新計算，且畫面會自動刷新。
-   **匯出功能**：一鍵匯出CSV檔案。

## 🛠️ 使用技術

-   **HTML5**：頁面結構與語義化標籤。
-   **CSS3**：使用 Flexbox 佈局與自定義樣式。
-   **JavaScript**：負責還款邏輯計算與動態 DOM 渲染。

## 📋 功能說明

### 系統輸入
-   **貸款金額**：以「萬元」為單位。
-   **貸款利率**：設定年利率百分比。
-   **貸款年限**：提供 3 至 40 年的選項。
-   **還款方式**：
    -   **本金平均攤還**：每月償還本金固定，利息隨剩餘本金遞減。
    -   **本息平均攤還**：每月償還總額固定（利用複利公式計算），方便記帳。

### 試算結果
-   **摘要表格**：顯示第一個還款金額、總還款金額及總利息支出。
-   **分期明細**：詳細列出每一期的月付額、攤還本金、攤還利息及貸款餘額，方便使用者追蹤還款進度。

### 💾儲存功能
-   **儲存CSV**：將每一期的月付額、攤還本金、攤還利息及貸款餘額，儲存成CSV格式。

## 📂 檔案結構

-   `index.html`：主要程式碼，包含結構、樣式與計算邏輯。
